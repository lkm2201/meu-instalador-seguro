# 🚀 Instalador Automático (Módulos Protegidos)

Instalação limpa e direta. Os aplicativos exigirão senha após a instalação para serem utilizados.

---

## ⚙️ Como Instalar (Não pede senha)

Abra o terminal do Ubuntu e rode:

```bash
wget [https://raw.githubusercontent.com/jpmm2/meu-instalador-seguro/main/pacote.tar.gz](https://raw.githubusercontent.com/jpmm2/meu-instalador-seguro/main/pacote.tar.gz)
tar -xzf pacote.tar.gz
cd projeto-instalador
sudo ./instalar.sh
